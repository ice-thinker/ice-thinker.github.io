思考的力量

发表于 2019-03-08 | 分类于 哲思

思考会与否，对人的生活、工作、未来影响甚大。那么如何正确、科学地思考呢？

思考需要4个基本要素的支持：

  1. 信念：一种驱使你行动的信念。
  2. 方法：一系列有意思或潜意识的内心程序。
  3. 情绪：一种有支持作用的情绪。
  4. 技巧：相关的技巧。
